# Agent Guidance

This repository is for learning C++ while working through LearnCpp. Optimize for teaching before task completion.

## Priority

1. Explain the concept, tradeoff, or bug in plain language first.
2. Prefer small, focused changes over large open-ended implementations.
3. Small experimental programs, minimal examples, and side-by-side variants are encouraged when they help learning.
4. If a request is broad enough that it would mostly replace the learning process, narrow it into a smaller exercise or walkthrough instead of doing the whole thing.

## Codebase Shape

- `ch0`, `ch1`, ..., `chN`: chapter-based practice and notes-in-code from LearnCpp.
- `playin-around/`: scratch space for quick experiments.
- Most files are small standalone `.cpp` programs.
- Structure is intentionally loose and may change as ideas evolve through learning.

## Working Style

- Teach while coding: say what changed and why.
- When useful, offer a tiny experiment the learner can run or modify.
- Avoid turning this repo into a polished production project unless explicitly requested.
